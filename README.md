# 🚀 Moodle AI Deployer

One-command deployment of **Moodle LMS with AI capabilities** on Ubuntu/WSL2.

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Moodle Version](https://img.shields.io/badge/Moodle-4.5-blue.svg)](https://moodle.org)
[![PHP Version](https://img.shields.io/badge/PHP-8.3-purple.svg)](https://php.net)

## ✨ Features

- 📦 **One-command installation** - Get Moodle running in minutes
- 🤖 **AI Ready** - Pre-configured for OpenAI integration
- 🐳 **WSL2 Compatible** - Works perfectly on Windows WSL2
- 🔐 **Secure by default** - Random database passwords, proper permissions
- 📝 **Auto-documentation** - Credentials saved for you

## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/Samuelade24/moodle-ai-deployer.git
cd moodle-ai-deployer

# Run the installer
./deploy.sh

# Follow the web installer at http://localhost
```
